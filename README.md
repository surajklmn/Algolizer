# Algolizer
This requires following dependencies to run:
```raylib
    cmake
    gcc or g++
```

```cmake -B build ```
Then cd to the build directory
```cd build```
run ```make```
```./Algolizer``` to run the program

It can also be build with this commanda

```g++ main.cpp -lraylib -lGL -lm -lpthread -ldl -lrt -lX11```
